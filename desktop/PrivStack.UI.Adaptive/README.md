# PrivStack.UI.Adaptive

Adaptive UI components for PrivStack plugins. Provides themed controls, editor integration, and layout primitives built on Avalonia.

## What's included

- **Themed controls** — Buttons, inputs, toggles, dropdowns, and badges that follow the active PrivStack theme
- **Block editor** — Rich text editing component with block-based content model
- **Graph view** — Knowledge graph visualization control
- **Layout primitives** — Responsive containers, split panes, and adaptive panels
- **Template engine** — `$for` loops, `$if` conditionals, and expression interpolation for dynamic UI trees

## Usage

```xml
<PackageReference Include="PrivStack.UI.Adaptive" Version="1.5.1" />
```

This package depends on [PrivStack.Sdk](https://www.nuget.org/packages/PrivStack.Sdk) and will pull it in automatically.
